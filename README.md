# 🛒 Laravel E-commerce App

## Features
- Product CRUD (Add, Edit, Delete)
- Image Upload
- Search & Pagination
- Cart System (Add, Remove, Checkout)
- Authentication (Login/Register)
- Responsive UI (Tailwind CSS)

## Tech Stack
- Laravel
- Blade
- Tailwind CSS
- SQLite/MySQL

## Setup
```bash
git clone <repo>
cd project
composer install
npm install
npm run dev
php artisan migrate
php artisan serve
